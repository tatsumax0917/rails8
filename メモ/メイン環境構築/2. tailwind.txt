※ tailwindcss-railsはスタンドアロン版だからpostcssのファイルや
tailwind.configなどは作成されず、必要なら自分で用意する


[Gemfile]
gem 'cssbundling-rails'
gem "tailwindcss-rails"
※ なんか上手くいかんかったら gem "tailwindcss-rails", "~> 3.0"にする

[command]
bundle install

[init]
bin/rails css:install:tailwind

[package.json]
"scripts": {
  "build": "yarn build:js",
  "build:js": "esbuild app/javascript/*.* --bundle --sourcemap --format=esm --outdir=app/assets/builds --public-path=/assets",
  "build:js:watch": "esbuild app/javascript/*.* --bundle --sourcemap --format=esm --outdir=app/assets/builds --public-path=/assets --watch",
  "build:css": "npx @tailwindcss/cli -i ./app/assets/stylesheets/application.tailwind.css -o ./app/assets/builds/tailwind.css --minify",
  "build:css:watch": "npx @tailwindcss/cli -i ./app/assets/stylesheets/application.tailwind.css -o ./app/assets/builds/tailwind.css --minify --watch"
},
※ tailwindはtailwind.cssとしてbuildされるように設定する
※ application.sass.scssとなってるかもだからその場合は.scssに調整＋実ファイルもapplication.scssに名前調整
※どっちでもい

[app/views/layouts/application.html.erb]

<%= stylesheet_link_tag "tailwind", "inter-font", "data-turbo-track": "reload" %>
<%= stylesheet_link_tag "application", "data-turbo-track": "reload" %>

※ ビルドされたtailwind.cssファイルを読み込むようにする

[Procfile.dev]
web: rails s -p 3000 -b 0.0.0.0
esbuild: yarn build:js:watch
tailwindcss: yarn build:css:watch
dartsass: bin/rails dartsass:watch

[tailwind.config.js]
※ 必要であれば手動でファイル作成して記述
module.exports = {
  content: [
    './app/views/**/*.html.erb',
    './app/helpers/**/*.rb',
    './app/assets/stylesheets/**/*.css',
    './app/javascript/**/*.{js,jsx}', // jsx使うかもしれんから
  ]
}
